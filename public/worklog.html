🌿 Connect your Google Sheet

This portfolio reads live data from your Google Sheet via SheetDB.
It takes 2 minutes to set up — no backend needed.

  1. Step 1: Go to sheetdb.io → Sign up free → "Create new API"
  2. Step 2: Paste your Google Sheet URL and create the API
  3. Step 3: Copy the API URL — looks like:
    https://sheetdb.io/api/v1/xxxxxxxxxxxxxxx
  4. Step 4: Open this HTML file, find SHEETDB_URL near the top of the <script> and replace it with your URL
  5. Step 5: Make sure your sheet tab is named Work log and has columns: Era, Title, Type, Date, Link

SheetDB free tier: 500 requests/month — plenty for a portfolio page.
